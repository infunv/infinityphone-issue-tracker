# InfinityPhones — Issue Tracker

Welcome to the official issue tracker for **InfinityPhones**, a premium communication system plugin for Minecraft RolePlay servers.

⚠️ **Note:** The core source code of this plugin is proprietary and closed-source. This repository does not contain any plugin code and is used **strictly for tracking bugs, compatibility issues, and feature requests**.

---

## 🐛 How to Report an Issue

If you found a bug or want to suggest a new feature, please follow these steps:

1. **Search Existing Issues:** Check the [Issues](../../issues) tab to see if your problem has already been reported.
2. **Open a New Issue:** If it's a new problem, click "New Issue" and fill out the details.
3. **Provide Information:** To help us fix the issue faster, please include:
   * Your server software and version (e.g., Purpur 1.20.4).
   * Exact steps to reproduce the bug.
   * Error logs from the console (use [Pastebin](https://pastebin.com) or code blocks).
   * Screenshots or videos (if applicable).
